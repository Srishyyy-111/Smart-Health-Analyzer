# Smart Health Analyzer

A Python-based data analytics project using Pandas to clean, analyze, and generate insights from health-related data.

## Overview
This project simulates real-world healthcare data containing missing values and abnormal readings.  
It performs data cleaning, feature engineering, and basic health risk classification.

## Key Features
- Handles missing values using column-wise mean
- Caps BMI values for standardization
- Identifies unhealthy individuals based on medical thresholds
- Generates descriptive statistics
- Exports cleaned data to CSV

## Technologies Used
- Python
- Pandas

## Concepts Applied
- DataFrames
- Data Cleaning
- Iteration & Conditional Logic
- Feature Engineering
- Descriptive Analytics

## Output
The final cleaned dataset is saved as a CSV file for further analysis or visualization.
